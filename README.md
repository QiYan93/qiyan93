## qiyan93
qiyan93个人文件杂货铺

[![Build Status](https://www.travis-ci.org/QiYan93/qiyan93.svg?branch=master)](https://www.travis-ci.org/QiYan93/qiyan93)

### 启动
```
npm install

npm start
```

### 说明

数据源是聚合数据
express后台渲染，模版是jade

nodejs写本站api，后台requests转发聚合api，返回json数据
